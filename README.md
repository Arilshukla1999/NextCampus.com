🌍 NextCampus

NextCampus is a modern web platform designed to help students explore universities, scholarships, and study abroad opportunities in a simple and interactive way.

🚀 Live Website

👉 https://next-campus-com.vercel.app/

---

✨ Overview

NextCampus simplifies the study abroad journey by providing:

- University discovery
- Scholarship filtering
- AI-powered guidance
- Clean and user-friendly interface

---

🎓 Features

University Explorer

- Browse top universities worldwide
- View key details like fees, rankings, and programs

Scholarship Filter

- Discover scholarships based on preferences
- Helps reduce financial barriers

AI Chat Assistant

- Suggests universities based on user needs
- Provides quick and helpful guidance

Modern UI/UX

- Clean, responsive design
- Smooth navigation
- Mobile-friendly experience

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Deployment: Vercel
- AI Integration: API-based chatbot
